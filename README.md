## About
Magnetic Field Visualizer is an interactive web visualizer that makes invisible magnetic fields visible by allowing users to place magnets and current-carrying wires.
It instantly displays field vectors and lines, helping users intuitively understand how magnetic fields behave and interact.
- Try it here: [link](https://takshsingh313.github.io/MagField-Visualizer/)

## Features

- Interactive Magnetic Sources : Users can place and manipulate magnetic dipoles and current-carrying wires to explore different field configurations.
- Real-Time Vector Field Visualization: Magnetic fields are displayed as arrows that show both direction and strength, updating instantly with user interaction.
- Magnetic Field Lines (Streamlines): Smooth field lines illustrate how magnetic fields flow and interact, helping users understand overall field behavior.
